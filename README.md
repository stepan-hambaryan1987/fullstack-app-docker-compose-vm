# Fullstack Application with Docker Compose

# Description
This project demonstrates a containerized fullstack application using Docker Compose.

The application includes:
- Frontend
- Backend
- MySQL database

# Technologies
- Docker
- Docker Compose
- MySQL

# Features
- Multi-container application
- Service communication via Docker network
- Persistent storage using volumes
- Healthcheck for database

# Architecture
Frontend → Backend → MySQL

# How to run

```bash
docker-compose up --build
